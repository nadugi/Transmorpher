# Transmorpher

Transmorpher is a client-side transmog system based on morphing for World of Warcraft WotLK 3.3.5a .

It lets you change visual appearance locally by morphing:
- Character model
- Equipped items
- Mounts
- Companion pets
- Hunter combat pets
- Weapon enchants
- Titles
- Time of day

## Version

Current release: **1.1.1**
